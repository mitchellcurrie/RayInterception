Ray Interception Calculator


Instructions:
- Run the application using the command line
- Use four command line arguments to represent the two screen coordinates in this order:
	- Ps1_x, Ps1_y, Ps2_x, Ps2_y
- The results are printed to the console
- The executable requires two files in the assets folder: "camera.json" and "green.obj"
